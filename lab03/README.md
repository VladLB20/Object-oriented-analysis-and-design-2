# Отчет о проделанной работе

## 1. Цель работы
Реализовать проект, который показывает статус доставки.

## 2. Диаграмма классов для паттерна
![Реализация паттерна](ДК.drawio.png)

State (абстрактный класс) интерфейс для всех конкретных состояний. (process, pay, allow_shipping, allow_delivery, deliver, cancel), а также метод name(). 

Accepted — начальное состояние «Принят».  Переходы: «В обработку» (process) или «Отмену» (cancel).\
ProcessState — «В обработке». Переходы: «Оплачен» (pay) или «Отменён» (cancel).\
Paid — «Оплачен».  Переходы: «Разрешена отгрузка» (allow_shipping) или «Отменён» (cancel).\
allowedShipping — «Разрешена отгрузка». Переходы: «Разрешена доставка» (allow_delivery) или «Отменён» (cancel).\
allowedDelivery — «Разрешена доставка». Переходы: «Доставлен» (deliver) или «Отменён» (cancel).\
stateDelivery — конечное состояние «Доставлен».\
canceleState — конечное состояние «Отменён».\
